# Autocomplete System

Autocomplete words with a given prefix.<br>
Implemented using an AVL tree.

## Running the program

To start, run the `main.py` file.

## Command-line interface (CLI)

The list of available command options:

| Command | Description |
| --- | --- |
| 1 | Insert a word |
| 2 | Search for a word |
| 3 | Autocomplete a word |
| 4 | Show all words |
| 0 | Show commands |
| -1 | Exit |
